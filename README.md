# yStnadard Toolbox

無料WordPressテーマ「yStandard」用機能拡張プラグイン

## 更新履歴

### v0.2.0
- [修正] 記事一覧ブロック タグ選択不具合修正
- [調整] 見出し編集機能　カスタム選択時は文字設定以外をクリア
- [調整] 見出し編集機能　吹き出しブロックの背景色を変えた際に高度な設定も同期する
